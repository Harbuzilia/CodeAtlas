# Project Context Navigation

## Purpose
Navigation for project-specific patterns. Customize this for each project.

---

## Quick Routes

| Intent | File |
|--------|------|
| Project patterns | `patterns.md` |

---

## Files

### `patterns.md`
- When: Any project-specific code
- Contains: Naming conventions, folder structure, API patterns, component patterns

---

## Customization

Add project-specific files:
- `api-patterns.md` — API endpoint patterns
- `components.md` — UI component patterns
- `database.md` — Database schema patterns
- `security.md` — Project security requirements

---

## Priority

Project context has highest priority — it overrides core standards when there's a conflict.
